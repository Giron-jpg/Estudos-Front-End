Criando formulários

Para criar um formulário usamos a tag form e dentro dela inserimos os elementos necessários. Para criar a caixa de digitação usamos a tag input que é acompanhada de mais uma informação que varia de acordo com o tipo de dado. É importante que todo input seja acompanhado de um label para que haja direcionamento entre o nome do elemento e o input. Ex.:


Alguns parâmetros:

action - cria um vínculo com um arquivo de outra linguagem (posteriormente associado a um script).

autocomplete (on/off) - mostra opções para completar uma caixa em um cadastro.

method - define o método de compartilhamento de envio de dados. Por padrão ele é definido como GET (limite máximo de 3000 letras), com isso os dados preenchidos no formulário são expostos na url (ex.: buscas no google - ajuda no compartilhamento). O método POST (Prefirido no envio de imagens) mascara as respostas do formulário na url porém ainda ficam acessíveis no dev tools > Network > nome.arquivo > headers. Para mais segurança devemos aprender sobre HTTPS
