iFrames

A efeito de comparação são estruturas similares a janelas, permitindo visualizar conteúdos de outros sites. Um exemplo é o método de incorporar (compartilhar) vídeos do youtube. Por padrão esse modelo possui estrutura inline e um tamanho de 300x150px.

Podemos utilizar o iframe para exibir algum conteúdo previamente carregado na mesma página. Para isso precisamos atribuir um name ao iframe e em seguida inserir no link um target para esse nome.

Segurança de iframes

Para evitar que sites permitam enviar dados para dentro de um iframe usamos o parâmetro sanbox com valor sandbox dentro do iframe. Isso permite que o iframe se torne invalido no envio de dados. Também é possível usar o valor allow-same-order que permite conteúdos de mesma origem; allow-forms permite uso de formulários; allow-script permite scripts.

Outro parâmetro é o referrerpolicy que se refere a politica de coleta de dados (usado principalmente por sites de anúncios), por padrão este parâmetro vem como "no-referrer-when-downgrade". Para evitar que quais quer dados sejam coletados podemos usar o valor no-referrer

Conteúdos em iframes

Com iframes podemos inserir elementos em nossa página de outros sites sem a necessidade de deter um vídeo ou imagem. Exemplos de seu uso são: inserção de vídeos do youtube, inserção de mapas do google maps (ou mapas ao vivo do waze), ou até mesmo documentos publicados na web; etc.
