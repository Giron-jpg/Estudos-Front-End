para escrever os comandos <p> e </p> é preciso usar o & seguido de lt
para o símbolo da esquerda e gt para o símbolo da direita

Na verdade essas abreviações são para: Maior que (Greather than) e menor que (Less Than)

O uso do & auxilia na inserção de qualquer tipo de simbolo e emoji. Importante ressaltar que ao final do comando dever constar o ; (ponto e vírgula). Exemplos:

® ©; ™ € $ Δ

Vamos adicionar alguns emojis com base no site emojipedia usando o código hexadecimal;

Usando links

Links externos

Para lidar com links externos usamos a tag a e inserindo a url em href, com ela também podemos associar os comandos target ="_blank" e rel= "external" para fazer com que o link acesse outra página

Exemplo Github

Links internos

Para inserir links internos também utilizamos o tag a "âncora", porém damos preferência a usar a mesma página

Links para download

Segue link para download teste do meu currículo por meio da tag a, com o parâmetro download e o type (o tipo é encontrado através do site iana.org): currículo

Para arquivo em zip usariamos no parametro download o nome do arquivo e no type "application/zip"

Estilos CSS

Existem 3 tipos de configurações para inserir estilos em uma página CSS. São eles:

  • Estilo inline: é inserido na mesma linha em que se destaca uma tag html.
  • Estilo Interno: é inserido no cabeçalho da página (head).
  • Estilo Externo: cria-se um arquivo de estilo css
